# Quiz - Lesson 009: Streaming Responses with Server-Sent Events (SSE)

1. **Which ConverseStream event block contains the text chunk generated by the model?**
   - A) `messageStart`
   - B) `contentBlockStart`
   - C) `contentBlockDelta`
   - D) `metadata`

2. **At what stage of the stream is the `metadata` token usage event emitted?**
   - A) At the very beginning
   - B) Mid-stream
   - C) At the very end of the stream
   - D) Every 5 seconds

3. **Why is using ConverseStream preferred over low-level InvokeModelWithResponseStream?**
   - A) It is cheaper to execute
   - B) It provides a standardized event output schema across all models, eliminating model-specific stream decoding wrappers
   - C) It supports larger context windows
   - D) It does not require IAM credentials

## Answer Key

1: C, 2: C, 3: B
